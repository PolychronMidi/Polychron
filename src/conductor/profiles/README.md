# conductor/profiles/

Static ensemble-configuration profiles (instrument ranges, ensemble
shapes) consumed by the conductor. Distinct from `src/composers/profiles/`,
which holds STYLE-FAMILY profiles (blues, chromatic, modal, ...); these
are ENSEMBLE profiles, not style profiles.

If a "profile" file describes how musicians are arranged (range,
section, soloist role), it belongs here. If it describes how notes
are chosen (style, mode, voicing), it belongs in composers/profiles/.
